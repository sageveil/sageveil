<h2 align="center">@sageveil/hyper</h2>

<p align="center">Sageveil colors for the Hyper terminal.</p>

## Install

```sh
hyper i @sageveil/hyper
```

Or add the plugin manually to `~/.hyper.js`:

```js
plugins: ['@sageveil/hyper']
```

## Build from the monorepo

```sh
pnpm nx run hyper:generate
```

The generated npm package is written to `dist/ports/hyper/`.

## Testing status

This port has not yet been tested in Hyper. Please report any issues.
